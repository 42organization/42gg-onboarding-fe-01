# 📝 ToDo List 만들기

## ✨ 주요 기능 소개
### ✏️ 할 일을 추가할 수 있습니다
- 오늘의 할일을 입력하고, 엔터키를 통해서 제출해서 새로운 할 일을 등록합니다.

### ✏️ 할 일을 삭제할 수 있습니다
- 삭제 버튼을 이용해서 삭제가능합니다.

- 수정버튼을 클릭하고, 내용을 수정 후 다시 수정버튼을 클릭하면 기존의 할일을 수정 가능합니다.
